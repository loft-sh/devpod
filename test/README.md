### Run the e2e tests

Make sure you have ginkgo installed on your local machine:
```
go get github.com/onsi/ginkgo/ginkgo
```

#### Run all e2e test suites
```
# Install ginkgo and run in this folder
ginkgo e2e/...
```

#### Run a specific e2e test suite
```
# Install ginkgo and run in this folder
ginkgo e2e/smoke
```
